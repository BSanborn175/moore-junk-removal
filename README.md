### Deployment (Same as LienGuard Waitlist)
1. Create GitHub repo: `moore-junk-removal`
2. Upload `index.html`
3. Enable GitHub Pages (Settings → Pages → main branch)
4. In IONOS: Add CNAME record → `www` pointing to `bsanborn175.github.io`
5. In IONOS: Add A records for root domain pointing to GitHub Pages IPs:
   - 185.199.108.153
   - 185.199.109.153
   - 185.199.110.153
   - 185.199.111.153
6. In GitHub Pages settings: Set custom domain to `moorecountyjunkremoval.com`
7. Enable HTTPS

### Google Business Profile (CRITICAL — Do This First)
1. Go to business.google.com
2. Create profile for "Moore County Junk Removal"
3. Category: "Junk Removal Service"
4. Add service area (Moore County and all towns)
5. Add photos (truck, before/after jobs)
6. Link to moorecountyjunkremoval.com
7. This gets you on Google Maps and local search results
